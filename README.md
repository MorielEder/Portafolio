## Acerca del Proyecto

Este portafolio ha sido diseñado para presentar proyectos realizados. El sitio web integra:

- ✅ Presentación profesional y habilidades técnicas
- ✅ Dashboard interactivos de Power BI integrados directamente
- ✅ Proyectos reales de análisis de datos con insights accionables
- ✅ Diseño responsive y accesible
- ✅ Optimizado para SEO y rendimiento

### Diseño Moderno

El diseño combina una paleta de colores profesional en tonos verdes (inspirada en datos y crecimiento) con una estructura limpia que prioriza la experiencia de usuario y la visualización de proyectos.

## Tecnologías Utilizadas

### Frontend
- **HTML5 Semántico** - Estructura accesible y bien organizada
- **CSS3 Moderno** - Variables CSS, Flexbox, Grid, animaciones
- **Google Fonts** - Tipografía Inter para mejor legibilidad
- **Font Awesome 6** - Iconografía profesional

### Visualización de Datos
- **Power BI** - Dashboards interactivos embebidos
- **DAX** - Fórmulas para cálculos avanzados
- **Power Query** - Transformación y limpieza de datos

### Bases de Datos
- **SQL Server** - Consultas y gestión de datos
- **PostgreSQL** - Base de datos para proyectos analíticos

### Herramientas
- **Git & GitHub** - Control de versiones
- **Google App Script** - Automatización de procesos
